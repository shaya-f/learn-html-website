The <head> Tag

The head tag contains everything that is not shown on the main content of the page. Rather, it contains what the page needs to use, including the title and links to other files for CSS or JavaScript

The head tag looks like this within what we've previously done

<!DOCTYPE html>

<html>

  <head>

    |

  </head>

</html>
