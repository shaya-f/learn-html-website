The Basics of HTML

<!DOCTYPE>

For any html document, the first thing that has to be done is defining the type of document

The tag for this is:

<!DOCTYPE html>


Once that's taken care of, the next thing is to use the html tag which will wrap the entire web page

The html tag however, needs to be closed once it has been opened. to do this, simply repeat the same tag but with a backslash directly after the first one

In all, it looks something like this:

<!DOCTYPE html>

<html>

  |

</html>

this concludes the very basics of html, we covered the very first steps into making a website
